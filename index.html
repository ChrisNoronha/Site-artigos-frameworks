Prós e Contras
‬‬‬‬‬‬‬‬‎ ‎ ‎‎ O AngularJS é um framework JavaScript desenvolvido pela Google para criação de aplicativos da web de página única (SPA - Single Page Applications). Ele fornece uma estrutura para desenvolver aplicativos web dinâmicos, onde as alterações na página ocorrem sem a necessidade de recarregar a página inteira.

‬‬‬‬‬‬‬‬‎ ‎ ‎‎ Segue o padrão MVC (Model-View-Controller) para separar a lógica da aplicação em três componentes principais: Model (Modelo): Representa os dados e o estado da aplicação. View (Visualização): É a camada de apresentação da aplicação, responsável pela exibição dos dados para o usuário. Controller (Controlador): Gerencia as interações entre o modelo e a visualização. Ele recebe as ações do usuário, atualiza o modelo e reflete essas alterações na visualização.

‬‬‬‬‬‬‬‬‎ ‎ ‎‎ Ele utiliza a sintaxe declarativa do HTML para adicionar funcionalidades dinâmicas às páginas. Ele estende o HTML com diretivas personalizadas, permitindo a criação de elementos interativos e reativos. As diretivas são atributos ou elementos HTML que informam ao AngularJS como manipular e exibir os dados.

‬‬‬‬‬‬‬‬‎ ‎ ‎‎ Uma das principais características é o two-way data binding, que significa que as alterações nos dados do modelo são automaticamente refletidas na visualização, e vice-versa. Isso simplifica o desenvolvimento de aplicativos interativos, pois elimina a necessidade de manipulação manual do DOM (Document Object Model).

‬‬‬‬‬‬‬‬‎ ‎ ‎‎ Além disso, oferece recursos adicionais, como injeção de dependência, validação de formulários, roteamento, animações e testabilidade facilitada. Ele também suporta a criação de componentes personalizados e permite a integração com outras bibliotecas e framework.

‬‬‬‬‬‬‬‬‎ ‎ ‎‎No entanto, é importante mencionar que o AngularJS foi substituído pelo Angular (também conhecido como Angular 2+). O Angular é uma versão totalmente reescrita do AngularJS e apresenta melhorias significativas de desempenho, modularidade e recursos adicionais. É recomendado utilizar o Angular em novos projetos, mas o AngularJS ainda é amplamente utilizado em projetos legados.
